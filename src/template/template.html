You can get phones by /phones or /phones?page=*&size=*&sort=* endpoint

If page, size or sort are not mentioned, they will be set by default
Sort values: title, age, price
Page by default = 1
Size by default = 16
Sort by default = age
Will return object with params:
  • phones
  • total
  • page
  • size

Get phone by /phones/:phoneId endpoint

Will return phone object

Get recommended phones by /phones/:phoneId/recommended endpoint

Will return object with array of 8 recommended phones

Get best price phones by /phones/discount endpoint

Will return object with array of 12 phones with the biggest sales

Get new phones by /phones/new endpoint

Will return object with array of 12 newest phones
