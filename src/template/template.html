You can get products by /products or /products?page=*&size=*&sortBy=*&productType=* endpoint

If page, size, sortBy or productType are not mentioned, they will be set by default
sortBy values: title, age, price
page by default = 1
size by default = 16
sortBy by default = age
productType by default = products
Will return object with params:
  • products
  • total
  • page
  • size
  • sort
  • category

Get product by /products/:productId endpoint

Will return product object

Get recommended products by /products/:productId/recommended endpoint

Will return object with array of 8 recommended products

Get best price products by /products/discount endpoint

Will return object with array of 12 products with the biggest sales

Get new products by /products/new endpoint

Will return object with array of 12 newest products
